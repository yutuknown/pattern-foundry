# Pattern Foundry v0.2.0

Release date: 2026-03-25

This release packages major marketplace distribution improvements, stricter output contracts, and full bundled engine assets for self-contained installs.

## Highlights

### Added
- Added `SHIP_UPDATE_GUIDE.md` with step-by-step ship commands for push, PR, tag, and install validation.
- Added `scripts/verify_packaged_plugin.py` to validate required engine files in packaged payloads.
- Added `scripts/sync_plugin_engine.py` and documented the maintainer sync flow in README.
- Bundled complete engine assets under `plugins/pattern-foundry/skills/pattern-foundry/engine/`.
- Added packaged references under `plugins/pattern-foundry/skills/pattern-foundry/references/`:
  - `mode-guide.md`
  - `output-contracts.md`
  - `evaluation-rubric.md`
- Added explicit completion status and comparison notes in `AUDIT_REPORT.md`.
- Added inspiration attribution in README acknowledgements.

### Changed
- Rewrote packaged `SKILL.md` to enforce mandatory output contracts across all modes, explicit assumptions handling, and stricter audit/page-generation response structure.
