# Pattern Foundry v0.1.0

We are thrilled to announce the first public release of **Pattern Foundry** (formerly *transferable-uiux-pattern-engine*).

## What is Pattern Foundry?

Pattern Foundry is a powerful [Claude Code](https://code.claude.com/docs/en/overview) skill designed to extract, abstract, and generate premium UI/UX interfaces. Instead of cloning a reference site, Pattern Foundry identifies the underlying *design system intelligence*—like bimodal border radii, spacing rhythm, interaction physics, and conversion architecture—and applies those transferable principles to build entirely original brands and templates.

## Key Features in v0.1.0

*   **Extraction Scripts (Python):** Robust Playwright-based scripts (`inspect_site.py`, `extract_tokens.py`) that crawl a reference site to capture semantic design tokens and DOM patterns.
*   **Design System Persistence:** A persistent `MASTER.md` file acts as the global source of truth, while minimal `pages/*.md` files handle page-specific overrides.
*   **Brand Adaptation Model:** Built-in rules to shift temperature, density, and tone to fit new industries (B2B SaaS, Healthcare, Fintech, etc.).
*   **Anti-Copy Guardrails:** Strict instructions to prevent source-brand leakage, ensuring all generated outputs are original and brand-safe.
*   **Sample Artifacts:** High-fidelity React and Tailwind CSS sample components, fully decoupled from the reference brand.

## Current Limitations

*   Extraction scripts currently require manual refinement of the generated `tokens.json` to assign specific roles to extracted colors.
*   Visual outputs (e.g., hover states) are CSS-driven and require a modern browser compatible with the output stack.
*   The current template samples are heavily optimized for React + Tailwind CSS; Vue and SvelteKit support is planned for future releases.

## Next Planned Improvements

*   Multi-site token fusion (combining principles from 3+ premium reference sites).
*   Automated accessibility auditing inside the extraction pipeline.
*   Direct export compatibility with Figma Tokens Studio.
