# Pattern Foundry v0.2.1

Release date: 2026-03-25

This release cleans up the repository structure, keeps release docs under source control, and clarifies the packaging workflow so the marketplace plugin mirrors the source engine.

## Highlights

### Added
- Introduced `docs/audits/` (audit history + rescan report) and `docs/releases/` (release notes, checklist, ship guide) directories.
- Saved a detailed deep-audit report and a follow-up “file-level rescan” status file for traceability.
- README now links directly to the release docs and explains where assets live (`assets/readme/*`).

### Changed
- README messaging now emphasizes prompt-first usage for marketplace installs and clearly distinguishes maintainer-only extraction scripts.
- CONTRIBUTING clarifies manual audit setup and codifies the sync/verify workflow for engine edits.
- Engine README + summary (source and packaged copies) now use neutral wording, note version `0.2.0`, and redact source-brand specifics.
- Release checklist was reset for v0.2.1 with explicit pre-release/launch/post-release steps to avoid missed actions.

### Fixed
- `.gitignore` entry for `.agent/` no longer contains corrupted null characters.
- References in release notes/checklists now point to the relocated docs paths (no more stale `release-notes/` or root audit references).
