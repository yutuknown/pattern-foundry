# Pattern Foundry Mode Guide (Packaged)

## NEW_BRAND_MODE
Use when creating a net-new branded experience.
Output emphasis: token adaptation + page architecture + CTA logic.

## AUDIT_MODE
Use when critiquing existing UI.
Output emphasis: evidence-backed scoring + prioritized fixes + implementation direction.

## PAGE_GEN_MODE
Use for generating one specific page.
Output emphasis: section blueprint + component map + build handoff.

## TEMPLATE_MODE
Use for reusable, brand-agnostic templates.
Output emphasis: placeholders + variant knobs + accessibility defaults.

## SYSTEM_EXPAND_MODE
Use to add features/pages to an existing design system.
Output emphasis: consistency constraints + minimal overrides + regression checks.
