# Output Contracts

These contracts are mandatory for deterministic outputs.

## Audit Output Contract
- 10-dimension score table
- Top 5 issues by impact
- Upgraded spec (layout + token + component changes)
- Implementation plan (phased)
- 60-minute quick wins

## Page Generation Output Contract
- Objective + target user
- Section blueprint
- Token map
- Component map
- Build handoff checklist

## Brand Output Contract
- Assumptions
- Token set
- IA and CTA rhythm
- Component/motion grammar
- Delivery checklist
