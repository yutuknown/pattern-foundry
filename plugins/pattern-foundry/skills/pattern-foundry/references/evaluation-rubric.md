# Evaluation Rubric

Score each from 1-10 with short evidence notes:

1. Originality
2. Hierarchy
3. Spacing
4. CTA clarity
5. Trust placement
6. Responsiveness
7. Accessibility
8. Component coherence
9. Motion restraint
10. Implementation cleanliness

## Minimum quality bar
- Every dimension >= 7/10
- Any score <7 must include a concrete corrective action.
