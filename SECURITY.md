# Security Policy

## Supported Versions

Currently, we only provide security updates for the latest major version.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within Pattern Foundry, please do not disclose it publicly. 
Instead, please send an email to the repository maintainers or open a private security advisory on GitHub. 

We will acknowledge receipt of your vulnerability report and strive to send you regular updates about our progress.
